Etiquetas Multimedia

Listado de etiquetas multimedia
Etiqueta Descripción Atributos Ejemplo
<img> Incrusta una imagen en el documento HTML.
  • src: ruta de la imagen.
  • alt: texto alternativo.
  • width / height: define tamaño.
<audio> Reproduce archivos de sonido o música.
  • src: ruta del archivo de audio.
  • controls: muestra controles de reproducción.
  • autoplay: inicia el audio automáticamente.
<video> Permite reproducir archivos de video directamente en la página.
  • src: ruta del video.
  • controls: muestra controles de reproducción.
  • poster: imagen previa antes de reproducir.
<figure> + <figcaption> Se usa para asociar una imagen, gráfico o contenido con una descripción o título.
  • id: identificador único.
  • class: clase CSS.
  • figcaption: texto descriptivo.
<source> Permite especificar múltiples archivos multimedia para un mismo elemento <audio> o <video>.
  • src: ruta del archivo.
  • type: tipo MIME (audio/mp3, video/mp4).
