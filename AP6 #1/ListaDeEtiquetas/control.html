Etiquetas de Control

Listado de etiquetas de control
Etiqueta Descripción Atributos Ejemplo
<form> Etiqueta principal para crear formularios interactivos.
  • action: URL donde se envían los datos.
  • method: método HTTP (GET o POST).
  • name: nombre del formulario.
<input> Permite capturar datos del usuario como texto, número, fecha o contraseña.
  • type: tipo de dato (text, password, number, etc.).
  • placeholder: texto guía dentro del campo.
  • required: campo obligatorio.
<select> + <option> Despliega un menú de opciones para elegir un valor.
  • name: nombre de la lista.
  • value: valor que se envía al seleccionar.
  • selected: marca la opción predeterminada.
<textarea> Permite ingresar texto en múltiples líneas, como comentarios o descripciones.
  • rows: número de filas visibles.
  • cols: ancho del área de texto.
  • placeholder: texto guía.
<button> Crea un botón que puede ejecutar acciones o enviar formularios.
  • type: tipo de acción (submit, reset, button).
  • onclick: acción al hacer clic.
  • class: clase de estilo.
